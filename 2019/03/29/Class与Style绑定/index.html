Class与Style绑定

发表于 2019-03-29 | 分类于 前端
  1. 操作元素的 class列表 和 内联样式 是数据绑定的一个常见需求。因为他们都是特性(attributes)，所以我们可以用 v-bind 处理他们——只需要通过表达式计算出 字符串结果 即可。
  2. 不过，字符串拼接麻烦且易错。因此，在将 v-bind 用于 class 和 style 时，Vue.js做了专门的增强。表达式结果除了 字符串 之外，还可以是 对象 或 数组
