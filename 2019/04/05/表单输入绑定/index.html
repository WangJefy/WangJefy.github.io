表单输入绑定(Form Input Bindings)

发表于 2019-04-05 | 分类于 前端

基础用法(Basic Usage)

  1. 你可以用 v-model 指令在表单 <input 、<textarea> 及 <select> 元素上创建双向数据绑定。
  2. 它会根据控件类型自动选取正确的方法来更新元素。
  3. 尽管有些神奇，但 v-model 本质上不过是语法糖。
  4. 它负责监听用户的输入事件以更新数据，并对一些极端场景进行一些特殊处理。
  5. v-model 会忽略所有表单元素的 value 、checked 、selected 特性(attributes)的初始值，而总是将Vue实例的数据作为数据来源。你应该
  6. 通过JavaScript在组件的 data 选项中声明初始值
